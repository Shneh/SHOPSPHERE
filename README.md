# 🛒 ShopSphere – Revolutionizing Your Cart Experience

“Discover new possibilities with every purchase.”

ShopSphere is a web-based online shopping cart system enhanced with motivational features like personalized offers, gamification (reward spins), and social sharing to encourage users to complete their purchases and reduce cart abandonment.

---

## 🚀 Live Demo

Frontend: [https://shopsphere.netlify.app](https://shopsphere.netlify.app)  
Backend API: [https://shopsphere-backend.onrender.com](https://shopsphere-backend.onrender.com)

---

## 🧰 Tech Stack

### Frontend:
- HTML, CSS, JavaScript (Vanilla)
- Responsive layout (mobile-first design)
- Hosted on **Netlify**

### Backend:
- Python with Flask
- RESTful APIs (CORS enabled)
- Hosted on **Render**

### Database:
- In-memory for now (can be extended to MongoDB or MySQL)

---

## ✨ Features

- ✅ Add items to cart
- ✅ Apply dynamic discounts
- ✅ Gamify with "Spin the Wheel" rewards 🎯
- ✅ Get AI-powered recommendations
- ✅ Share cart via WhatsApp or social platforms 📤
- ✅ Clean, responsive design
